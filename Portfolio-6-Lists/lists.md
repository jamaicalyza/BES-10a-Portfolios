# PORTFOLIO 5: STRINGS
## Multi-Code String Encoder System (Morse, Caesar Cipher, and Binary Translator)
This portfolio features a Multi-Code String Encoder System that allows users to convert
text into different encoded formats such as Morse Code, Caesar Cipher, and Binary Code,
as well as decode them back into readable text. The project demonstrates how string
manipulation can be applied across multiple encoding systems using core programming
concepts including Strings, Functions, Conditionals, and Iterations to build a structured
and interactive translation tool
